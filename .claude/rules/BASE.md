# Multi-agent workflow cycle

Non-trivial / multi-step work only: **architect** (analyze, design, split) → **pm** (dispatch to **developer**, bg-update the three docs) → **architect** (validate conventions, web-search latest stable) → **architect + ml-engineer** (pick the better option). Repeat until done.

Skip for trivial / single-file edits, quick fixes, lookups — do them directly (KISS). Match orchestration to task size; don't spawn agents when a direct edit suffices.
